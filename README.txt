HOW TO USE:

1️⃣ Install Python from https://www.python.org/downloads/ (if not already installed)
2️⃣ Open Command Prompt or Terminal
3️⃣ Navigate to the folder and run: pip install -r requirements.txt
4️⃣ Start the tool: streamlit run credit_assessment_tool.py

All your uploaded files and reports will be saved inside the 'Retailers' folder automatically.
